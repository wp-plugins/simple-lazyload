== Description ==
This plugin automatically copy image's src value to file attribute, replace src value with a blank image's url before showing, when the page is loaded, lazyload js will decide to load the images' actual content automatically, only when user wants to see them.　　本插件实现真实的图片迟加载功效，自动保存、替换图片的真实地址，只有当用户真实需要看到时，图片才会向服务器去请求内容，否着它是一张空白图片，对服务器没有负担。本插件可以与auto-highslide插件配合使用，效果更佳。当然你也可以使用另一个超强组合插件(http://blog.brunoxu.info/auto-lazyload-and-auto-highslide/)来取代它俩的功能。


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `simple-lazyload` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress Background.
